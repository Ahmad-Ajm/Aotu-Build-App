# Features Map – Project Feature Inventory
## 1. الميزات الرئيسية
- **إنشاء السيرة الذاتية**: إمكانية إدخال المعلومات وتنظيمها.
- **نظام تسجيل**: للمستخدمين لإضافة وتعديل السير الذاتية.
- **عرض السير الذاتية**: إمكانية عرض جميع السير الذاتية.
- **تصدير السير الذاتية**: مع إخفاء البيانات الحساسة.
- **بحث وتصفية**: لتسهيل الوصول إلى السير الذاتية.
