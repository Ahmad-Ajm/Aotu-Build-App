# Specification: FEAT-CV-CREATION

## المتطلبات الوظيفية
1. إنشاء واجهة لإنشاء السيرة الذاتية مقسمة إلى أقسام (معلومات شخصية، التعليم، الخبرات ...)
2. إمكانية إضافة أو تعديل أو حذف أي قسم.
3. إمكانية إعادة ترتيب الأقسام
4. يجب تحقق صحّة المدخلات كلما تم التغيير
5. حفظ السيرة تلقائياً عند كل تعديل

## المتطلبات غير الوظيفية
- دعم اللغة العربية والإنجليزية
- حاجة لاستجابة واجهة تفاعل سريعة UX
- حماية البيانات عند النقل والتخزين

## السيناريوهات القوية
- مستخدم يدخل بيانات كاملة ويحفظ ويعود لاستكمال
- تعديل قسم دون أن يفقد البيانات الأخرى

## API
- endpoint: /api/profile/cv
- أنواع العمليات: POST (إنشاء)، PUT (تعديل)، GET (جلب بيانات المستخدم)

## قاعدة البيانات
- CVCategories: Id, CVId, Name, Order
- CVItems: Id, CVCategoryId, FieldName, Value

## الواجهة (Angular)
- صفحات forms بالـ ReactiveForms
- التحكم بالسحب والإفلات للأقسام
- إشعارات toast للنجاح والفشل
