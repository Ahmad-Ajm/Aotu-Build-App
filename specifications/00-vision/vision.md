# رؤية المشروع (Vision)
- **المشكلة**: صعوبة كتابة السيرة الذاتية (CV) وإيصالها للمهتمين.
- **الحل**: إنشاء CV بشكل تلقائي من خلال إدخال المعلومات، وتنظيمها في قاعدة بيانات، مع إمكانية البحث والتصفية.
- **القيمة**: تسهيل عملية إنشاء السيرة الذاتية وتوفير وسيلة فعالة لتخزينها وعرضها.
- **الجمهور المستهدف**: الأفراد الذين يحتاجون إلى إنشاء سير ذاتية.
