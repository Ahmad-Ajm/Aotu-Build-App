# رؤية المشروع (Vision)
- **المشكلة**: صعوبة كتابة السيرة الذاتية (CV) وتوصيلها للمهتمين.
- **الحل**: إنشاء سير ذاتية تلقائيًا بناءً على المعلومات المدخلة، وتنظيمها في قاعدة بيانات، مما يسهل البحث والتصفية.
- **القيمة**: تسهيل عملية إنشاء وتوزيع السير الذاتية، مما يوفر الوقت والجهد للمستخدمين. مؤشرات قياس القيمة: عدد السير الذاتية المنشأة، معدل الاستخدام، رضا المستخدمين.
- **النطاق الزمني**: النسخة الأولية (MVP) ستتضمن الوظائف الأساسية المذكورة، مع إمكانية إضافة ميزات لاحقة.
- **الجمهور المستهدف**: الأفراد الذين يحتاجون إلى إنشاء سير ذاتية بسهولة، مثل الطلاب والباحثين عن عمل.
