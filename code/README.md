# مشروع إنشاء السيرة الذاتية

## نظرة عامة
هذا المشروع هو تطبيق ويب لإنشاء وإدارة السير الذاتية باستخدام:
- **Backend**: .NET 8 مع ABP Framework
- **Frontend**: Angular 17
- **قاعدة البيانات**: PostgreSQL أو SQL Server

## الميزات الرئيسية
1. **إنشاء السيرة الذاتية (FEAT-CV-CREATION)**
   - إدخال المعلومات الشخصية
   - إدارة الخبرات العملية
   - إدارة التعليم والمؤهلات
   - إدارة المهارات
   - معاينة مباشرة
   - تصدير إلى PDF وWord

2. **تسجيل المستخدمين (FEAT-USER-REGISTRATION)**
   - تسجيل مستخدمين جدد
   - تسجيل الدخول
   - إدارة الحسابات

## هيكل المشروع

### Backend (.NET 8 + ABP)
```
src/
├── Domain/           # الكيانات والنطاق
├── Application/      # الخدمات و DTOs
├── Infrastructure/   # التنفيذات
└── Web/             # واجهات برمجة التطبيقات
```

### Frontend (Angular 17)
```
src/app/
├── cv/              # مكونات السيرة الذاتية
├── auth/            # المصادقة
├── shared/          # مكونات مشتركة
└── core/            # الخدمات الأساسية
```

## بدء التشغيل
سيتم إضافة تعليمات بدء التشغيل بعد إنشاء الملفات الأساسية.
