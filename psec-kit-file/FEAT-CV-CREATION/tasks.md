### 2. تطوير واجهات برمجة التطبيقات (Backend)
- [ ] **TASK-CV-004**: تطوير واجهات إدارة السير الذاتية
  - GET /api/cv - الحصول على جميع السير الذاتية للمستخدم
  - GET /api/cv/{id} - الحصول على سيرة ذاتية محددة
  - POST /api/cv - إنشاء سيرة ذاتية جديدة
  - PUT /api/cv/{id} - تحديث سيرة ذاتية
  - DELETE /api/cv/{id} - حذف سيرة ذاتية
  - POST /api/cv/{id}/duplicate - نسخ سيرة ذاتية

- [ ] **TASK-CV-005**: تطوير واجهات إدارة الأقسام
  - GET /api/cv/{cvId}/sections - الحصول على جميع الأقسام
  - POST /api/cv/{cvId}/sections - إضافة قسم جديد
  - PUT /api/cv/{cvId}/sections/{id} - تحديث قسم
  - DELETE /api/cv/{cvId}/sections/{id} - حذف قسم
  - PUT /api/cv/{cvId}/sections/reorder - إعادة ترتيب الأقسام

- [ ] **TASK-CV-006**: تطوير واجهات التصدير
  - GET /api/cv/{id}/export/pdf - تصدير إلى PDF
  - GET /api/cv/{id}/export/docx - تصدير إلى DOCX
  - GET /api/cv/{id}/export/html - تصدير إلى HTML

- [ ] **TASK-CV-007**: تطوير واجهات إدارة القوالب
  - GET /api/templates - الحصول على جميع القوالب
  - GET /api/templates/{id} - الحصول على قالب محدد
  - POST /api/templates - إنشاء قالب جديد (للمسؤولين)
  - PUT /api/templates/{id} - تحديث قالب (للمسؤولين)
