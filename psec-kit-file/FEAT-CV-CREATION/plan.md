# خطة التنفيذ - إنشاء السيرة الذاتية (FEAT-CV-CREATION)

## نظرة عامة
إنشاء نظام يسمح للمستخدمين بإنشاء سيرة ذاتية (CV) بسهولة من خلال واجهة مستخدم بسيطة وتخزين البيانات في قاعدة بيانات.

## المكونات الرئيسية

### 1. واجهة المستخدم (Frontend - Angular 17)
- **صفحة إنشاء CV**: نموذج تفاعلي لجمع معلومات السيرة الذاتية
- **مكونات Angular**:
  - CV Creation Component
  - CV Form Component
  - CV Preview Component
  - CV List Component
- **تصميم واجهة المستخدم**:
  - نموذج متعدد الخطوات (Multi-step form)
  - معاينة مباشرة (Live preview)
  - تصميم متجاوب (Responsive design)

### 2. الخلفية (Backend - .NET 8 + ABP Framework)
- **الخدمات**:
  - CV Service (لإدارة عمليات CRUD)
  - CV Template Service (لإدارة قوالب السير الذاتية)
  - Export Service (لتصدير السير الذاتية)
- **الكيانات (Entities)**:
  - CV (البيانات الأساسية)
  - Education (التعليم)
  - Experience (الخبرة العملية)
  - Skill (المهارات)
  - ContactInfo (معلومات الاتصال)

### 3. قاعدة البيانات (PostgreSQL)
- **الجداول**:
  - CVs (السير الذاتية)
  - Educations (التعليم)
  - Experiences (الخبرات)
  - Skills (المهارات)
  - Contacts (معلومات الاتصال)
  - Templates (قوالب السير الذاتية)

## تدفق العمل

### الخطوة 1: جمع المعلومات
1. المستخدم يصل إلى صفحة إنشاء CV
2. يملأ المعلومات الأساسية (الاسم، العنوان، البريد الإلكتروني)
3. يضيف التعليم (المؤهلات الأكاديمية)
4. يضيف الخبرات العملية
5. يضيف المهارات
6. يضيف معلومات الاتصال

### الخطوة 2: المعاينة والتخصيص
1. معاينة السيرة الذاتية بشكل مباشر
2. اختيار قالب من القوالب المتاحة
3. تخصيص الألوان والخطوط
4. إعادة ترتيب الأقسام

### الخطوة 3: الحفظ والمشاركة
1. حفظ السيرة الذاتية في قاعدة البيانات
2. توليد رابط مشاركة فريد
3. خيارات التصدير (PDF، Word)
4. إدارة الخصوصية (عام/خاص)
